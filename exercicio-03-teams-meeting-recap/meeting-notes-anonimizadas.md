# Notas anonimizadas da reunião

## Contexto

Reunião técnica para alinhamento de uma integração B2B entre sistemas corporativos, utilizando API Gateway e documentação técnica no padrão Swagger/OpenAPI.

## Participantes

- Time técnico interno
- Time técnico parceiro
- Representante de negócio
- Responsável pela arquitetura da integração

> Observação: nomes reais, empresas, e-mails e URLs foram removidos para preservar a confidencialidade e evitar exposição de dados sensíveis.

## Pontos discutidos

- Definição do fluxo de consumo da API.
- Necessidade de disponibilização da documentação Swagger/OpenAPI.
- Alinhamento sobre autenticação e autorização.
- Definição das responsabilidades entre o sistema consumidor e o sistema fornecedor.
- Validação dos principais cenários de consumo.
- Discussão sobre tratamento de erros e retorno das APIs.
- Necessidade de ambiente de homologação para testes.
- Próximos passos para validação técnica da integração.
