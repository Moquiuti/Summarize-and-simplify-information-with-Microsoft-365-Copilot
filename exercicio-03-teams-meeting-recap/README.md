# Exercício 03 - Destacando decisões e ações em reuniões do Teams com Copilot

## Objetivo

Utilizar o Microsoft Copilot no Teams para resumir uma reunião, identificar decisões importantes, destacar pontos de atenção e organizar itens de ação.

## Contexto utilizado

Foi utilizado um cenário fictício/anônimo baseado em uma reunião técnica de integração B2B entre sistemas, envolvendo API Gateway, documentação Swagger, responsabilidades entre as partes e próximos passos técnicos.

## Cuidados com LGPD

As informações foram anonimizadas. Não foram incluídos nomes de pessoas, empresas, clientes, e-mails, URLs internas, dados sensíveis, prints de reunião ou qualquer informação que permita identificação real dos envolvidos.

## Resultado esperado

Gerar um resumo profissional da reunião, contendo:

- Principais decisões;
- Itens de ação;
- Responsáveis genéricos;
- Próximos passos;
- Pontos de atenção.
