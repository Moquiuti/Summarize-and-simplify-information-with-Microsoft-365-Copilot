# Summarize-and-simplify-information-with-Microsoft-365-Copilot

Repositório criado para armazenar as atividades práticas realizadas durante o curso de Microsoft 365 Copilot.

As atividades envolvem o uso do Copilot para resumir informações, identificar dados relevantes, analisar tendências e gerar respostas estruturadas a partir de arquivos do PowerPoint e Excel.
