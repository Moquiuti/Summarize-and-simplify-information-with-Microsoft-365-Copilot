# Exercício 04 - Preparação da semana com Copilot no Outlook

## Objetivo

Utilizar o Microsoft Copilot no Outlook para resumir uma comunicação por e-mail, identificar informações importantes, organizar pendências e apoiar a preparação da semana de trabalho.

A atividade simula o uso do Copilot para analisar um e-mail relacionado à liberação de uma versão de software para homologação.

## Contexto

Neste exercício, foi utilizado um e-mail técnico enviado a um cliente informando que uma nova versão do sistema estava pronta para homologação.

O conteúdo original foi anonimizado antes de ser documentado neste repositório, substituindo nomes reais de clientes, ambientes e referências internas por nomes fictícios.

## Arquivos deste exercício

- `email-anonimizado.md`  
  Contém o e-mail utilizado como base para a atividade, com dados sensíveis removidos ou substituídos.

- `prompt-utilizado.md`  
  Contém o prompt criado para orientar o Copilot no Outlook a resumir o e-mail e destacar informações importantes.

- `resposta-final.md`  
  Contém o resultado esperado da análise, estruturado com objetivo do e-mail, principais informações, itens de ação, prazo, riscos e próximos passos.

## Prompt utilizado

```text
Summarize this email conversation.

I need to prepare for the week and follow up on a software release that was sent to the client for homologation.

Focus on the main release information, pending actions, deadline, deployment artifacts, key changes, risks, and next steps.

Create a professional summary in Portuguese using the following format:

1. Email objective
2. Main information
3. Action items
4. Deadline
5. Risks or important notes
6. Recommended next steps

Do not include real client names, personal data, internal URLs, emails, credentials, or sensitive business information.
