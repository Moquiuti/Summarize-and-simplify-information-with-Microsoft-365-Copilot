# Exercício 01 - Resumindo informações no PowerPoint com Copilot

## Objetivo

Utilizar o Microsoft Copilot no PowerPoint para resumir informações importantes de uma apresentação, identificando os principais pontos de negócio de forma clara e profissional.

## Arquivo utilizado

- Mystic Spice Premium Chai Market Analysis Presentation.pptx

## Prompt utilizado

Summarize slides 5-10 in this PowerPoint presentation for my boss. The summary should include the main points before a client meeting. Please format the answer as a professional bulleted list, focusing only on the key business information.

## Resultado

O Copilot foi utilizado para resumir os slides da apresentação, destacando os principais pontos sobre o produto Mystic Spice Premium Chai Tea, seus benefícios, oportunidade de mercado, canais de distribuição e estratégia comercial.
